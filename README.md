## Build

#### Prerequisites
- Rust (w/ Cargo)

Once cloned, rename the `.env.example` into `.env`.

Then follow the [SETUP.md](./SETUP.md) file for instructions on how to setup.

Build the binary:
```sh
cargo build --release
```
